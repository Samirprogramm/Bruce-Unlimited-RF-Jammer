# Bruce Unlimited RF Jammer

Personal experimental fork related to Bruce firmware.

## Important Safety Notice

RF jamming and intentional radio interference can be illegal and dangerous outside of very limited authorized test environments. This repository must not be used to interfere with public, private, emergency, commercial or consumer radio systems.

Use only in a controlled, shielded and lawful lab environment where you own the equipment and have explicit permission to test.

## What This README Provides

This README documents the repository boundary and safety expectations. It intentionally does not provide operational instructions for radio interference.

## Allowed Use

- Legal firmware research.
- Documentation cleanup.
- Defensive hardware learning.
- Controlled lab analysis with permission.

## Not Allowed

- Interfering with other people's devices.
- Testing against public networks or public spectrum.
- Disrupting communications.
- Using the project in a way that violates local law.

## Disclaimer

The repository owner is responsible for complying with all applicable laws and regulations.
